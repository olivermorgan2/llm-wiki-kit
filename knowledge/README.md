# knowledge/ — curated project knowledge layer

A **lightweight, human-curated** knowledge layer for `llm-wiki-kit`,
layered on top of the Claude Code Workflow Kit's tool-maintained `design/`
artifacts. It holds the distilled understanding that orients a human or an
agent fast — the brief, live risks, open questions, and the decision log —
as curated summaries and pointers, **not** transcript dumps or generated
output.

Start here:

- **[`index.md`](index.md)** — live front door: every file, the key
  `design/` pointers, the current phase, and the next action.
- **[`SCHEMA.md`](SCHEMA.md)** — the layer's conventions: file roles, entry
  formats, status vocabularies, and what belongs here vs in `design/`,
  `notes/`, or `prompts/`.

`design/` is authoritative; `knowledge/` links into it rather than
duplicating it.
